== Description ==

Este plugin simples é um widget que exibe uma lista de top de jogos casuais da BigFish em sua barra lateral widgetized na linguagem Português só (.pt, .com.br).
É muito útil se você deseja criar um site para diferentes idiomas e quer usar todos os tops de jogos para diferentes idiomas.

ENGLISH:
This simple plugin is a widget that displays a list of top casual games from BigFish on your widgetized sidebar on Portuguese language only (.pt, .com.br).
It's very useable if you want to create one site for different languages and want to use all tops of games for different languages.

== Installation ==

1. Carregar pasta widget contém 'widget-top-bfg-games-PT.php' para o '/wp-content/plugins/' diretório.
2. Ative o plugin através do menu 'Plugins' no WordPress.
3. Use seu 'Presentation'/'Sidebar Widgets' secção para colocar o widget e configurá-lo.

ENGLISH:
1. Upload widget folder contains `widget-top-bfg-games-PT.php` to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use your 'Presentation'/'Sidebar Widgets' section to place the widget and to configure it.

== Configuration ==

* Widget título: o título do widget
* Número de jogos para mostrar: Número de jogos top você gostaria de ser exibido (1-10)
* Tipo de jogos para mostrar: Jogos Tipos que você gostaria de ser exibido (jogos de PC / Jogos Online)
* Texto e Imagens Align: Alinhamento da lista de jogos que você gostaria de ser exibida (default tema / esquerda / centro / direita)
* Show Bullets Lista: É preciso mostrar balas lista (use estilo tema)
* Show Posição Jogos: É preciso mostrar a posição na lista de jogos de topo
* Show Imagens Jogos: É preciso mostrar imagens de jogos, além de título do jogo
* Tipo de Imagens: Imagens Tipo de você gostaria de ser exibido (pequeno 60x40 / default 80x80 / featured 175x150 / subfeatured 175x150)
* Show Texto Jogos: É preciso mostrar texto jogos, além de título do jogo ea imagem
* O que Acima: Qual Acima você gostaria de ser exibido - texto ou imagem
* Seu login BigFishGames: Você pode deixar este campo em branco, widget pode trabalhar sem login, ou o uso de login correto.
* O seu canal do cliente BigFishGames: Nome de Domínio será usado por padrão. Deixe este campo em branco, para não usar os canais. Alfa numéricos.

ENGLISH:
* Widget title: the title of the widget
* Number of Games to Display: Number of top games you would like to be displayed (1-10)
* Games Type to Display: Games Types you would like to be displayed (PC games / Online Games)
* Text and Images Align: Align of games list you would like to be displayed (theme default / left / center / right)
* Show List Bullets: Is need to show list bullets (use theme style)
* Show Games Position: Is need to show games position in top list 
* Show Games Images: Is need to show games images in addition to game title
* Images Type: Images Type you would like to be displayed (small 60x40 / default 80x80 / featured 175x150 / subfeatured 175x150)
* Show Games Text: Is need to show games text in addition to game title and image
* What Above: What Above you would like to be displayed - Text or Image
* Your BigFishGames login: You can leave this field empty, widget can work without login, or use correct login.
* Your BigFishGames client channel: Domain name will be used by default. Leave this field empty, to not use channels. Alpha numeric only.
